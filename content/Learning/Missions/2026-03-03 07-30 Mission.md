# Mission: Container Security Quick Audit with Trivy

**Objective:** In 45 minutes, scan one Docker image for vulnerabilities and produce a short remediation plan.

## Steps
1. Pick an image you use often (example: `python:3.12-slim`).
2. Install Trivy (if needed): `sudo dnf install trivy` or use official install script.
3. Run scan:
   ```bash
   trivy image --severity HIGH,CRITICAL --format table python:3.12-slim
   ```
4. Export machine-readable report:
   ```bash
   trivy image --severity HIGH,CRITICAL --format json -o trivy-report.json python:3.12-slim
   ```
5. Identify top 3 issues (CVEs + affected package + fix status).
6. Propose one safer alternative image tag and re-scan it.
7. Write a 5-8 bullet remediation summary.

## Expected Output Artifact
- `trivy-report.json`
- `container-security-summary.md` (top findings + recommended image/tag + next actions)

## Stretch Challenge
Integrate Trivy into a CI check (GitHub Actions or GitLab CI) that fails only on **CRITICAL** vulnerabilities.
