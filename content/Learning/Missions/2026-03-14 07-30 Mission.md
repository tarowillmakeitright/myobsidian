# Mission: Build a Tiny RAG Evaluator with Local Embeddings

Tags: #learning #mission #ai #devops #science  
Link: [[Home]]

## Objective
In 30–60 minutes, create a minimal script that compares retrieval quality for two embedding models on a small document set.

## Exact Steps
1. Create a folder `rag-mini-eval` and add 8–12 short docs (markdown/text) on mixed topics.
2. Pick two embedding models (e.g., `all-MiniLM-L6-v2` and `bge-small-en-v1.5`).
3. Write a Python script to:
   - chunk docs,
   - embed chunks with each model,
   - run 5 test queries,
   - return top-3 chunks per query.
4. Make a simple scoring table (0/1 relevance) for each returned chunk.
5. Compute Recall@3 for each model and print a comparison summary.
6. Save results to `results.md`.

## Expected Output Artifact
- `rag_eval.py` (or notebook)
- `results.md` with query-by-query outputs + final Recall@3 comparison

## Stretch Challenge
Containerize it with a `Dockerfile` and run the same evaluation reproducibly in one command (`docker run ...`).
