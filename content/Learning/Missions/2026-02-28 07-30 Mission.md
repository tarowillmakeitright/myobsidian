# 2026-02-28 Mission

[[Home]]

## Mission Title
**Build a Tiny RAG Assistant with Local Embeddings**

## Objective
In 30–60 minutes, create a minimal Retrieval-Augmented Generation (RAG) workflow that can answer questions from a small custom knowledge file.

## Exact Steps
1. Create a short knowledge file (`notes.txt`) with 10–20 lines on one topic you care about.
2. Generate embeddings for each chunk (use any familiar tool/library: e.g., Python + sentence-transformers/OpenAI embeddings).
3. Store vectors in a lightweight index (FAISS, Chroma, or even in-memory cosine similarity).
4. Ask 3 test questions and retrieve top relevant chunks for each.
5. Produce final answers using retrieved context + your LLM of choice.
6. Record what worked and one failure case.

## Expected Output Artifact
A small repo/folder containing:
- `notes.txt`
- `rag_demo.py` or `rag_demo.ipynb`
- `results.md` with 3 Q&A examples and retrieved context snippets

## Stretch Challenge
Add a simple CLI command (`python rag_demo.py "your question"`) that returns the answer and prints the top-2 retrieved chunks with scores.
